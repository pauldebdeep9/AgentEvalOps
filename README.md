# AgentEvalOps

A framework for evaluating AI agents in operational settings — helping teams measure, monitor, and improve agent performance across real-world tasks.

## Overview

AgentEvalOps provides tooling and best practices for:

- **Evaluating** AI agent outputs against defined quality criteria
- **Tracking** agent performance metrics over time
- **Comparing** agent versions to detect regressions or improvements
- **Operationalizing** evaluation pipelines in CI/CD workflows

## Getting Started

### Prerequisites

- Python 3.9 or higher
- Git

### Installation

Clone the repository:

```bash
git clone https://github.com/pauldebdeep9/AgentEvalOps.git
cd AgentEvalOps
```

Install dependencies (coming soon — `requirements.txt` will be added as the project grows):

```bash
pip install -r requirements.txt
```

### Usage

```bash
# Run an evaluation
python evaluate.py --config config.yaml
```
