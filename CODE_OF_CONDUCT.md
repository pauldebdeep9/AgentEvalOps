# Code of conduct

This project follows the principle of respectful, constructive participation.

- Be welcoming and kind to contributors of all backgrounds and experience levels.
- Critique ideas, not people.
- Accept constructive feedback gracefully.
- Focus on what is best for the project and its users.

Harassment, discrimination, or personal attacks are not acceptable and will
result in removal from the project.

For concerns, open a GitHub issue or contact the maintainers privately via
GitHub Security Advisories.
